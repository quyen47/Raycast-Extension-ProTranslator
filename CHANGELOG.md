# ProTranslator Changelog

## [Initial Release] - 2024-05-30

### Added
- **Translator**: Core command to refine, translate, change tone, and continue text using AI.
- **Refine Inline**: A silent, background command to magically rewrite and replace selected text on the fly.
- **Practice Vocabulary**: A robust flashcard system to save and review learned words.
- **Quick Add Word**: Instantly look up AI definitions for unknown words and save them to vocabulary.
- **History**: Automatically save all AI text completions for easy access and re-copying.
- Multi-provider support: Easily switch between Gemini, OpenAI, Groq, DeepSeek, or any custom OpenAI-compatible API endpoints.
- Auto-execute mode for seamless translation directly upon hovering over options in the Translator command.
- Smart quote stripping algorithm to preserve or clean punctuation around single words natively.
