# ProTranslator for Raycast

ProTranslator is a powerful AI-driven Raycast extension designed to act as your ultimate writing assistant and language learning hub. It seamlessly integrates into your workflow to fix grammar, translate text, and help you learn vocabulary using spaced repetition.

## Features

- **✨ Improve Selected Text**: Grab any selected text across your Mac and instantly fix grammar, paraphrase, change tone, or continue writing.
- **⚡ Fix Grammar Inline**: A silent, fast command that fixes the grammar of your selected text and pastes it right back, replacing the original text instantly.
- **🌍 Refine & Translate**: Refine your English/Vietnamese text and translate it beautifully.
- **🧠 Flashcards & Practice**: Save any translation or corrected grammar directly to your Flashcards. Use the built-in spaced repetition system (SRS) to practice vocabulary every day.
- **🕒 History**: Keep track of all your AI queries and text improvements.
- **🛠 Custom Prompts**: Add your own customized AI prompts directly in the extension preferences.

## Supported Providers

You can use any of the following AI providers:
- **Google Gemini** (Default)
- **OpenAI** (ChatGPT)
- **DeepSeek**
- **Groq**
- **Any Custom OpenAI-compatible Provider**

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the extension in Raycast.
4. Go to **Raycast Settings > Extensions > ProTranslator** to add your API Key for your preferred provider.

## Usage

- Select text in any app (e.g., Safari, Notes, Word).
- Open Raycast and search for `Improve Selected Text` or `Fix Grammar Inline`.
- The extension will automatically read your selected text or your clipboard if accessibility permissions are restricted.
